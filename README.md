# Air Draw (MediaPipe HTML Project)

Air Draw is a browser-based drawing application that uses hand tracking to allow users to draw in the air using their fingers.

## Features
- Real-time hand tracking
- Draw using finger gestures
- Runs directly in the browser
- No installation required

## Tech Stack
- HTML
- JavaScript
- MediaPipe

## How to Run
1. Open `air_draw.html` in a browser
2. Allow camera access
3. Start drawing in the air

## Note
Works best on Chrome with camera permissions enabled.
